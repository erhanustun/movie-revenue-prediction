# Movie Revenue Prediction

Can we predict whether a movie will be a box office success?

## Project Overview

In this project, I built a machine learning model to predict whether a movie will be a high-revenue success using the TMDB dataset.

## Key Steps

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Training (Logistic Regression & Random Forest)
- Model Evaluation (Accuracy & ROC-AUC)

## Results

- Logistic Regression → Accuracy: 88%
- Random Forest → Accuracy: 89%
- ROC-AUC: ~0.93

## Key Insights

- Budget and popularity are the most important factors
- Genre has an impact, but is not dominant
- Movie success is multi-factor driven

## Dataset

TMDB Movie Dataset (Kaggle)

## Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

---

## Conclusion

Movie success is not determined by a single factor, but by a combination of budget, popularity, and audience engagement.
